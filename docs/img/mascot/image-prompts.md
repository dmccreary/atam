# Image Prompts: Dex the Robot

These are self-contained prompts for generating or regenerating each mascot pose.
Every prompt includes the full base character description so they can be used
independently. Re-anchor to `character-sheet.md` before editing any prompt.

---

## Base Description (embedded in every prompt below)

> A flat vector cartoon illustration of Dex the Robot, a friendly pedagogical
> mascot for a database architecture and ATAM textbook. Dex is a small, compact,
> rounded robot with a steel-blue (#4682B4) body and dark-orange (#E65100) accent
> details. Dex has a small antenna topped with a tiny database cylinder, an orange
> chest panel with a simple bar-chart icon, and a compact tool belt holding a
> wrench and a stylus. Dex's face shows two glowing LED dot-eyes forming an upward
> arc smile and a small rectangular mouth display with a warm pixel smile.
> Style: flat vector cartoon, clean outlines, minimal shading, transparent
> background. No text in image. Suitable for embedding in educational content at
> icon size (90px).
